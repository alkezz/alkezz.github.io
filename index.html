Work

Bopify is a music streaming service that takes inspiration from Spotify

Built using a Python/Flask Backend. SQLAlchemy was used for devloping the Backend and PostgreSQL for production. The Frontend was built using a combination of React, Redux, JSX, and HTML/CSS.

Features include:

  1. Account creation
  2. Playlist creation
  3. Music playback
  4. Image uploading using AWS
  5. Ability to follow playlists and users
  6. Liking songs

Enhance is an art portfolio sharing application built with Behance's features in mind

This application was my first app built with a team of engineers which gave me experience working in a Git environment while also honing my skills in pair programming

Enhance was built using a Python/Flask Backend with SQLAlchemy to handle the database and we used React, Redux, JSX for the Frontend

Features include:

  1. Account creation
  2. Portfolio creation
  3. Image uploading using AWS
  4. Following users
  5. The ability to like portfolios
  6. Leaving comments on portfolios

Groundbnb is a vacation home rental service modeled after Airbnb.

Built using a JavaScript/Express.js Backend with Sequelize to manage the development database, PostgreSQL was used for database management in production. React, Redux, and JSX were used to build out the Frontend

This was my first application I created, with it came the opportunity to grow as a Software Engineer. Looking back at this application after completing Bopify I was shocked at how barebones it was. With that in mind I did a giant overhaul on all my components to make the code easier to read and more efficient. I also used Material UI to create Modals for my components, such as the login/signup and creating a spot, and Google Maps API to show where the spot is located in the world. I am continuing to work on it but I can't say I'm not proud of the progress so far!

Features include:

  1. Account creation
  2. Spot creation/edit/deletion
  3. Image uploading using AWS
  4. Booking a spot with validations to avoid date overlap
  5. Leaving a review for a spot you've booked in the past, editing review, deleting review
